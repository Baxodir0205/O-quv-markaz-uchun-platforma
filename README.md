CaseEdu — O'quv markazlari uchun innovatsion platforma
CaseEdu — bu o'quv jarayonini avtomatlashtirish, o'qituvchi va talaba o'rtasidagi muloqotni osonlashtirish hamda ta'lim sifatini nazorat qilish uchun yaratilgan zamonaviy LMS platformasi.

🚀 Asosiy funksiyalar
Platforma 3 xil foydalanuvchi roli uchun optimallashtirilgan:

1. Admin Paneli (Boshqaruv markazi)
Foydalanuvchilar boshqaruvi: Talaba va o'qituvchilarni ro'yxatga olish va tahrirlash.

Guruhlar: Yangi o'quv guruhlarini shakllantirish va talabalarni biriktirish.

Monitoring: Tizimdagi umumiy faollikni statistik ko'rinishda kuzatish.

Ogohlantirishlar: Guruhsiz qolgan talabalar yoki bo'sh guruhlar haqida avtomatik bildirishnomalar.

2. O'qituvchi Paneli (Teacher Dashboard)
Vazifalar boshqaruvi: Talabalarga uy vazifalarini yuklash va ularni tekshirish.

Baholash tizimi: Topshiriqlarga ball qo'yish va qayta aloqa (feedback) berish.

Test va Case Study: Bilimlarni sinash uchun testlar va real holatlar (cases) yaratish.

Statistika: O'z guruhlari va talabalari ko'rsatkichlarini kuzatish.

3. Talaba Paneli (Student Dashboard)
O'quv materiallari: Darslar va materiallar bilan tanishish.

Vazifalar: Topshiriqlarni onlayn yuborish va baholarni ko'rish.

Bilimni sinash: Testlar va Case Study topshiriqlarini yechish.

Natijalar: O'zlashtirish reytingini shaxsiy kabinet orqali kuzatib borish.

🛠 Texnologiyalar
Backend: Python (Django Framework)

Frontend: HTML5, CSS3, JavaScript (Bootstrap/Tailwind)

Database: SQLite (Lokal) / PostgreSQL (Production)

Version Control: Git & GitHub
